21. 一质量为*m*的物体悬于一条轻绳的一端，绳另一端绕在一轮轴的轴上，如图所示．轴水平且垂直于轮轴面，其半径为*r*，整个装置架在光滑的固定轴承之上．当物体从静止释放后，在时间*t*内下降了一段距离*S*．试求整个轮轴的转动惯量(用*m*、*r*、*t*和*S*表示)．

![](images/5e383d1d45b7de4b9fa9143e4e9ea41713e055a9d72ac00614059c40393f341c.jpg)
<!-- ANSWER -->
J = mr<sup>2</sup>(gt<sup>2</sup>/(2S) - 1)
<!-- EXPLANATION -->
设绳子对物体(或绳子对轮轴)的拉力为*T*，则根据牛顿运动定律和转动定律得：
*mg* - *T* = *ma* ①
*T r* = *Jβ* ②
由运动学关系有：*a* = *rβ* ③
由①、②、③式解得：*J* = *m*(*g* - *a*)*r*<sup>2</sup> / *a* ④
又根据已知条件 *v*<sub>0</sub> = 0，*S* = (1/2)*at*<sup>2</sup>，*a* = 2*S* / *t*<sup>2</sup> ⑤
将⑤式代入④式得：*J* = *mr*<sup>2</sup>(*gt*<sup>2</sup>/(2*S*) - 1)

24. 用波长为500 nm (1 nm=10<sup>-9</sup> m)的单色光垂直照射到由两块光学平玻璃构成的空气劈形膜上．在观察反射光的干涉现象中，距劈形膜棱边<em>l</em> = 1.56 cm的<em>A</em>处是从棱边算起的第四条暗条纹中心．
(1) 求此空气劈形膜的劈尖角*θ*；
(2) 改用600 nm的单色光垂直照射到此劈尖上仍观察反射光的干涉条纹，*A*处是明条纹还是暗条纹？
(3) 在第(2)问的情形从棱边到*A*处的范围内共有几条明纹？几条暗纹？
<!-- ANSWER -->
(1) θ = 4.8×10<sup>-5</sup> rad；
(2) *A*处是明条纹；
(3) 共有三条明纹，三条暗纹．
<!-- EXPLANATION -->
(1) 棱边处是第一条暗纹中心，在膜厚度为*e*<sub>2</sub> = λ/2处是第二条暗纹中心，依此可知第四条暗纹中心处，即*A*处膜厚度 *e*<sub>4</sub> = 3λ/2．
由几何关系 *e*<sub>4</sub> = *l*·*θ*，得 θ = *e*<sub>4</sub>/*l* = 3λ/(2*l*) = 3×500×10<sup>-9</sup>/(2×1.56×10<sup>-2</sup>) = 4.8×10<sup>-5</sup> rad．
(2) *A*处膜厚为 *e*<sub>4</sub> = 3×500/2 nm = 750 nm．
对于λ' = 600 nm的光，连同附加光程差，在*A*处两反射光的光程差为 δ = 2*e*<sub>4</sub> + λ'/2 = 2×750 + 300 = 1800 nm = 3λ'，它是半波长的偶数倍，所以*A*处是明纹．
(3) 棱边处仍是暗纹，*A*处是第三条明纹，所以从棱边到*A*处共有三条明纹，三条暗纹．
